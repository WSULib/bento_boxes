Setup:

Step 1: Setup your development environment for PHP (for example WAMP - if you are using Windows)

Step 2: Copy the "BentoBox_demo" folder to your web server’s root directory

Step 3: Enter your credentials in the config file (BentoBox_demo/Config.xml)

Step 4: Start the demo (eg: http://localhost/BentoBox_demo )

Using the Demo

Enter the query in the search box
You will see 9 Bento Boxes (Books, Scholarly Articles etc), each with its own search criteria which is predefined in the Config file
You will also see 3 Widgets (Twitter, Flickr, WorldCat)
You can hover over a record for a quick view
You can click on a record title to go to the detailed record
You can click on a box title to see all results for that box
You can click on ‘View all results’ to see all the results
In a results page, you can use expanders, limiters, search modes, facets etc

Configuration

Configuration is done using BentoBox_Demo/Config.xml

CommonSearchParametersForEachBentoBox : This stores the search criteria that is common to all the bento boxes

Guest : If 'y' the session is designated as a guest session. If 'n', it is a regular session.

FullResultsSearchParameters : This stores the search criteria that is applied when you want to go to any results page

Hover: This stores all data elements that will be displayed in the hover

Bentobox: This stores the configuration details for each bento box and also the search criteria defined for that box

To add a new bento box: simply add a new <BentoBox> element

Widgets: This stores the widget related configuration details

Code Basics

Authentication: By calling Authenticate method with UserID / Password (using HTTP POST)

Create Session: By calling CreateSession method with Profile and the x-authenticationToken HTTP header

Search: By calling Search method with the search criteria

Detailed record: By calling Retrieve method with the AN and Database ID
